github 添加密钥实现无密码操作
   https://blog.csdn.net/wanglei_storage/article/details/53258804

一、生成ssh密钥对
    当然上传之前肯定要自己先生成ssh key了。
    windows的用户可以下载xshell之类的工具来生成；
    linux就直接输命令:ssh-keygen -t rsa    #一直回车下去，不输入密码!
二、查看公钥
    密钥对生成完成后存放于当前用户 ~/.ssh 目录中，查看 id_rsa.pub
